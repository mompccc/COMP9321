1). Prerequisite
The project is built on python 3.6 with flask, and some external module also imported. Users need to install all those packages for running the webpage robust without any error.

Using python setup tool to install external module:
	
	pip3 install flask
	pip3 install flask_restplus
	pip3 install sklearn
	pip3 install numpy
	pip3 install pandas

Make sure you have install all package above before running the program.

2). Download all files from Bitbucket
Change the directory to project:
	
	cd project

If you want to browse our website with designed UI and see all functions, run the following command:
	
	python3 project.py

If you want to use the RESTful API, run the following command:

	python3 restful.py

3). Browse the webpage
Just using browser to browse the website, and URL is:
	
	Http://127.0.0.1:5000
